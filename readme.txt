#################################    HOWTO    ##################################

1) read my article (all is explained inside), freely available at:
       * http://arxiv.org/abs/1407.4374
       * https://hal.archives-ouvertes.fr/hal-01024788

2) clone kali (or download it if you do not use Git):
       git clone https://github.com/arnaudporet/kali.git

3) open the file example.go

The algorithm is tested with Go version go1.6 linux/amd64 (Arch Linux).

How to get Go:
    https://golang.org/
    https://golang.org/doc/install

Most Linux distributions provide Go in their official repositories. For example:
    golang (Ubuntu)
    go (Arch Linux)
