How to:

    1) read my article (preprint version available at
    http://hal.archives-ouvertes.fr/hal-01024788 and
    http://arxiv.org/abs/1407.4374), at least the Method section

    2) clone the project in your home directory ~/

    3) open the template file ~/kali-targ/example_network.f95

The example network is a published boolean model of the mammalian cell
cycle [1].
