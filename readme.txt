How to:

    1) read my article, freely available at
    http://hal.archives-ouvertes.fr/hal-01024788 and
    http://arxiv.org/abs/1407.4374

    2) clone the project in your home directory:
    cd ~/ && git clone https://github.com/arnaudporet/kali-targ.git
    (or download it if you do not have git, but I strongly encourage you to use
    git)

    3) with you favorite text editor, such as Sublime Text, open the template
    file: ~/kali-targ/example_network.f95

The example network is a published boolean model of the mammalian cell
cycle [1].
